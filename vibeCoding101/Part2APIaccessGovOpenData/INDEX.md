## 🛠️ Technical Details

### API Endpoint
```
https://rt.data.gov.hk/v1/transport/mtr/getSchedule.php
```

### Required Parameters
- `line` - MTR line code (e.g., TML, ISL, KTL)
- `sta` - Station code (e.g., TUM, CEN, MOK)

### Optional Parameters
- `lang` - Language (EN, TC, SC)

### Response Format
JSON with train schedules, times, platforms, and delay status
